# Types of machine learning on the basic of supervision

1.Supervised
    >>Regression
    >>Classification
2.Unsupervised (Branching off from Supervised)
    >>Clustering
    >>Dimensionality Reduction
    >>Anomaly Detection
    >>Association
3.Semi-Supervised
4.Reinforcement
## Machine Learning Types Chart

| Category        | Subtype/Technique       | Description                                  | Examples                               |
|-----------------|-------------------------|----------------------------------------------|---------------------------------------|
| **Supervised**  | Regression              | Predicting continuous values.               | House Price Prediction, Sales Forecasting |
|                 | Classification          | Categorizing data into discrete classes.      | Spam Detection, Image Recognition       |
| **Unsupervised**| Clustering              | Grouping similar data points together.        | Customer Segmentation, Anomaly Detection|
|                 | Dimensionality Reduction | Reducing the number of features while preserving important information.| PCA (Principal Component Analysis)     |
|                 | Anomaly Detection        | Identifying unusual patterns or outliers.      | Fraud Detection, Network Intrusion       |
|                 | Association              | Discovering relationships between variables.    | Market Basket Analysis                   |
| **Semi-Supervised** | (Often a Hybrid Approach)| Utilizing both labeled and unlabeled data.     | Text Classification with limited labels |
| **Reinforcement** | (Agent/Environment Interaction)  | Learning through trial and error to maximize reward.| Game Playing, Robotics               |
